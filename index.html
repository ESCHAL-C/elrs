eLRS Front End Demo

This eLRS tool can accept coordinates in four different formats:

  • Lat, Lon
  • RouteID (Route Name), Reference Marker, and Displacement
  • RouteID (Route Name), And Distance from Origin
  • Control Section And Mile Point Measure

When multiple roads are returned, buttons appear to step through them.



Query by Lat Lon

Example (Texas City, TX):

Lat = 29.39780972
Lon = -94.98759004




Query by Reference Marker

Example (Refugio, TX):

RouteID = US0077-KG
ReferenceMarker = 622
Displacement = 0.065




Query by DFO

Example (Diboll, TX):

RouteID = FM1818-KG
DFO = 1.606




Query by Control Section

Example (Corsicana, TX):

ControlSectionNumber = 012201
MilePointMeasure = 2.394

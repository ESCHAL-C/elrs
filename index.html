eLRS Front End Demo

This eLRS tool can accept coordinates in four different formats:

  • Lat, Lon
  • RouteID (Route Name), Reference Marker, and Displacement
  • RouteID (Route Name), And Distance from Origin
  • CSJ And Mile Point Measure


Query by Lat Lon

Example:

Lat = 29.39780972
Lon = -94.98759004



Query by Route Name

Example 1:

RouteID = IH0010-KG
ReferenceMarker = 476
Displacement = 4


Example 2:

RouteID = IH0010-KG
DFO = .5



Query by CSJ

Example:

ControlSectionNumber = LY7609
MilePointMeasure = .2
